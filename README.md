# Автоматическая оценка мотивационных анкет
Модель для автоматической оценки эссэ кандидатов

## Задача
В каждой анкете кандидат отвечает на 3 вопроса. За каждый ответ вручную выставляется оценка от 0 до 3 баллов. Итоговый балл — это сумма трех оценок (от 0 до 9). Кандидат проходит отбор, если суммарно набрал 6 баллов и более.
В выборке для обучения 651 строка (train), для проверки — 150 строк (val).

## Как это устроена модель

* Признаки: Тексты кодируются через TF-IDF по словам и по буквам (кусочки по 3-4 символа для защиты от опечаток). Дополнительно считаются 9 базовых параметров: длина текста, количество слов, наличие цифр, флаг пропуска ответа и т.д.
* Алгоритм: Для каждого вопроса обучается свой ансамбль из линейной регрессии (Ridge, вес 0.4) и градиентного бустинга (GradientBoostingRegressor, вес 0.6).
* Валидация: Качество проверяется через Nested K-Fold (5 внешних фолдов, 3 внутренних для подбора параметров). Это защищает модель от переобучения на маленьком датасете и исключает утечку данных.
* Метрики: Основная метрика — квадратичная каппа Коэна (QWK), которая стандартно применяется для оценки эссе. Дополнительно отслеживаются MAE, RMSE и F1-score для финального решения (прошел/не прошел).

## Структура репозитория

* essay_scorer_rost.ipynb — ноутбук с анализом данных, экспериментами и графиками.
* essay_scorer.py — чистый скрипт для запуска обучения и инференса.
* val_predictions.xlsx — готовый файл с предсказаниями для валидационной выборки.
* requirements.txt — список необходимых библиотек.

## Запуск проекта

* pip install -r requirements.txt
* python essay_scorer.py
