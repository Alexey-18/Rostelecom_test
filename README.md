# Автоматическая оценка мотивационных анкет

Тестовое задание: обучить модель, которая выставляет оценки мотивационным письмам кандидатов на образовательные программы.

## Задача

Каждая анкета содержит ответы на 3 вопроса. Менеджер вручную ставит баллы от 0 до 3 за каждый вопрос. Итоговый балл — сумма трёх оценок (0–9). Проходной порог для отбора — **6 баллов**.

Данные: 651 train / 151 val.

## Подход

**Признаки**
- TF-IDF на символьных n-граммах (3–4) — хорошо работает для русского, устойчив к опечаткам
- TF-IDF на словесных уни-граммах
- 9 ручных признаков: длина текста, число слов, лексическое разнообразие, наличие цифр, флаг пропуска и др.

**Модель**
- Ансамбль: Ridge × 0.4 + Gradient Boosting × 0.6
- Отдельная модель на каждый из трёх вопросов

**Валидация**
- Nested K-Fold (outer=5, inner=3) — выбран из-за небольшого размера датасета
- Внешний цикл даёт несмещённую оценку, внутренний — честный подбор гиперпараметров

**Метрики**
- QWK (Quadratic Weighted Kappa) — основная, стандарт для задач essay scoring
- MAE, RMSE, F1 (бинарный отбор ≥6) — вспомогательные

## Запуск

```bash
pip install -r requirements.txt
python essay_scorer.py
```

Файл с данными `tests.xlsx` должен лежать рядом со скриптом.

**Включить BERT** (опционально, нужен интернет для загрузки весов):

```bash
pip install transformers torch
```

В `essay_scorer.py` поставить `USE_BERT = True` и выбрать модель:
- `cointegrated/rubert-tiny2` — ~30 MB, CPU, быстро
- `DeepPavlov/rubert-base-cased` — ~700 MB, лучше на GPU
